# IT Automation using PowerShell

## 📌 Overview
This project focuses on automating user account management tasks in a Windows environment using PowerShell scripting.

## ❗ Problem
In IT support, user account creation and role assignment are repetitive, time-consuming, and prone to human errors.

## 💡 Solution
This project automates:
- User account creation
- Password handling
- Administrator role assignment

## ⚙️ Features
- Dynamic user input
- Secure password conversion
- Error handling using try-catch
- Automated role assignment

## 🛠 Tools Used
- PowerShell
- Windows OS

## 📊 Outcome
This automation reduces manual workload, improves efficiency, and minimizes errors in IT operations.

## 🔗 Project Files
- `user-automation.ps1` → Main automation script

## 🎯 Future Improvements
- Integration with Active Directory
- Logging system for tracking user creation
- GUI-based interface for ease of use
